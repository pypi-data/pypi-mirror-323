
## 🔄 3. Gerador de PDF Avançado

### 🎯 Introdução
Crie documentos profissionais facilmente! 📊

### 📦 Instalação
1. 🔧 **Setup**:
   - 🐍 Instale Python
   - 📚 Instale FPDF:
```bash
pip install fpdf
```

### 💻 Uso
1. 🚀 Execute:
```bash
python gerador_pdf.py
```

2. 📝 Digite:
   - 📂 Nome do arquivo
   - 📋 Título do documento
   - 📑 Seções e conteúdo
   - ✅ "sair" para finalizar
