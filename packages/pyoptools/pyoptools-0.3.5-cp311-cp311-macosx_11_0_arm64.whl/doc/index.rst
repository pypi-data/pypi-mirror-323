Welcome to pyOpTools's documentation!
=====================================

.. note::
   This documentation is in a very alpha state at the moment. Use with care.

The pyOpTools are a set of packages that allow the simulation of optical
systems. It is being used to develop the
`freecad-pyoptools <https://github.com/cihologramas/freecad-pyoptools>`_ 
workbench for `freecad <http://freecad.org>`_.
